# newScraper


**newScraper** is an app that will scrape news from the New York Times site.  

newScraper Link: https://desolate-sierra-28029.herokuapp.com/

The app uses the following tech:

* node.js server
* express - web framework for node.js
* body-parser - parse incoming request bodies
* path - node.js utility for working with file paths
* express-handlebars - page templates and rendering
* mongoose - object modeler for node.js
* cheerio - parse/traverse/manipulte DOM - used to scrape data
* request - simplified HTTP request client
